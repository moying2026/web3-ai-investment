# 项目总则 — Web3 AI 投资决策系统

基于 Binance Web3 市场的 AI 投资决策系统，通过数据采集、AI 评估、模拟交易、实盘交易实现自动化的 Web3 一级市场新币投资。

## 五、铁律

| 规则 | 说明 |
|:--|:--|
| ❌ 不许可随意生成文件 | 所有文件必须在文档中定义 |
| ❌ 不许可文件名带版本号 | 版本写在文件头部 |
| ✅ 所有文件必须记录 | 路径、名称、用途必须在文档中体现 |
| ✅ 每个文档必须声明依赖 | 前置依赖 + 下游依赖 |

| ✅ 预算控制 | 总预算 $10,000 / 单笔 $100 / 持币上限 1,000 |
| ✅ 代码改动必须测试 | 后端改完 → 测试主管验收；前端改完 → 桌面自动化助理截图 |

## 六、核心原则

### 原则 1：分层阅读

新成员接手必须从顶层逐层向下：总则 → 概要 → 索引 → 架构文档

### 原则 2：每个文档声明依赖

前置依赖 + 下游依赖，缺一不可

### 原则 3：每个文档一个职责

不把所有信息塞进一个文件

### 原则 4：模拟盘与实盘统一

代码逻辑完全一样，唯一区别是 is_simulated 字段。切换实盘只改 is_simulated=0。

### 原则 5：使用 Binance 官方工具

交易使用 `baw` CLI（Binance Agentic Wallet），不自己造轮子。
